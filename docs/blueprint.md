# **App Name**: Mwaliko App

## Core Features:

- User Authentication & Authorization: Secure login and registration for all user roles (Super Admin, Event Admin, Scanner Staff, Reception Officer) with role-based access control to ensure appropriate permissions for system functionalities.
- Event Creation & Management: Event organizers can create, configure, and manage their events, defining details like event name, type, date, time, venue, guest capacity, cover image, and multiple entry phases or checkpoints.
- Guest Management & QR Generation: Organizers can add guests manually or import lists (CSV, Excel) with details such as name, phone, custom categories, companions, seat numbers, and special notes. The system automatically generates unique, secure QR codes containing encrypted guest and event identifiers for each attendee.
- QR Scanning & Validation System: A mobile-first, fast scanner interface for staff to scan guest QR codes using the device camera. The system instantly validates guest access (valid, already scanned, invalid, blocked) at specific checkpoints, displays guest details, and allows manual override with appropriate permissions, while storing temporary scan records offline if needed.
- AI-Assisted WhatsApp Invitation Content & Template Generation: An AI-powered tool to help organizers generate personalized invitation messages and design visually appealing invitation card templates, including custom branding, QR code placement, and thematic elements, based on event type and desired tone, sending these directly to guests via WhatsApp.
- RSVP Management: Guests can confirm their attendance (Attending, Not Attending, Maybe, Attending with Companions) via a personalized web link, and organizers can view real-time RSVP analytics in their dashboard.
- Dashboard & Reporting: A comprehensive dashboard for event organizers displaying key metrics like total guests, RSVP statuses, scanned guests, and attendance by category/checkpoint. Basic reports are exportable (PDF, Excel, CSV). Super Admins have an additional panel for overall system management.
- Multi-language Support: Full localization for the application interface and content, supporting both English and Swahili to cater to a broader user base.

## Style Guidelines:

- Primary Color: A deep, sophisticated Charcoal Grey (#2C3E50) to convey elegance and professionalism.
- Accent Color: A rich, warm Gold (#D4AF37) for highlights, calls to action, and luxury elements, providing a strong contrast.
- Background Color: A subtle, warm Off-White (#F8F8F2) for clean, spacious backgrounds, enhancing readability and a premium feel.
- Headline Font: 'Playfair Display' for its elegant, high-contrast serif design, perfect for luxurious titles and headings.
- Body Font: 'Open Sans' for excellent readability across various screens, maintaining a clean and professional appearance that complements the headline font.
- Clean, minimalist line icons, potentially with subtle metallic gradients or a consistent gold accent for interactive states, ensuring clarity and a touch of luxury.
- Mobile-first, spacious card-based design with ample whitespace, emphasizing a premium aesthetic and intuitive navigation. Implement responsive design, dual dark/light modes, and a prominent language switcher for English and Swahili.
- Refined and smooth micro-interactions and transitions, designed to enhance user experience without feeling intrusive, providing elegant feedback and a sense of polish.
